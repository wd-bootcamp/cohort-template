## Aufgaben

1. Create this file structure:

```
notes
├── released
│   └── public
│       ├── announcement1.txt
│       └── announcement2.txt
└── unreleased
    ├── announcement3.txt
    └── private
        ├── notes1.txt
        └── notes2.txt
```

2. Then change it to this structure:

```
notes
├── private
│   ├── notes1.txt
│   └── notes2.txt
└── public
    ├── released
    │   ├── announcement1.txt
    │   └── announcement2.txt
    └── unreleased
        └── announcement3.txt
```

3. Play a bash shell game

- Go to [Command Line Heroes](https://www.redhat.com/en/command-line-heroes/bash/index.html)
  and play the game.
